Example: jQuery unobtrusive validation

This example shows how you can use the adaptor for jquery unobtrusive validation. Here are a few cards you can try:

  • 371449635398431 (Amex)
  • 5105105105105100 (Mastercard)
  • 4111111111111111 (Visa)
Something like '41111' is not a valid credit card, but works on the default jQuery validation library.
